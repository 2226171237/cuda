从表可以看出，最终在GPU上优化结果比cpu串行程序快了14倍左右。在进行cuda优化时，考虑到大部分时间都是在进行cpu与gpu间的数据交互，我们要减小这种数据交互，以此来减小时间开销；同时要充分利用GPU的计算资源，增大block的数目，增加每个SM常驻线程快的数目来最大化并行性，提高运算速度。
